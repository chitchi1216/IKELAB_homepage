# IKELAB2026 — Brand Identity Guidelines

> **IKELAB2026 は自然科学系研究サークルです。**
> 本書は、そのロゴタイプに関するデザイン哲学・制作仕様・運用ルールをまとめた公式ガイドラインです。

---

## 01. デザインコンセプト — 「INVARIANT / 恒常」

自然科学は、世界の中に「不変量（invariant）」を探す営みです。光速、重力定数、エントロピーの増大、物理法則の対称性。ありとあらゆる変換を受けてもなお、変わらず成り立つ量。その静かな確かさこそが、科学という知的態度の核心にあります。

IKELAB2026 のロゴはこの思想を視覚化したものです。**一つの書体・一つの色・一つの形**。文脈がどれほど変わろうとも、ロゴそのものは同じ顔で立ち続けます。名刺の片隅にあっても、スライドの見出しになっても、Tシャツに刷られても、Webサイトの最上部に置かれても — 姿を変えない。これは「観測者の定数」としてのロゴです。

素材は、スイス・タイポグラフィの系譜に連なるネオ・グロテスク体「Instrument Sans Bold」です。Helvetica と同じ血統を持ちながら、微かに人文主義的な温かみをたたえた中立書体。そこに **−20 の光学トラッキング**を施し、十文字（I・K・E・L・A・B・2・0・2・6）を一つの塊として凝縮しました。一文字ずつ浮き立たず、読点のないひと息の呼吸として読まれます。

色は黒、もしくは白、それ以外を選びません。これは貧しさではなく、禁欲です。自然科学の論文が黒いインクで書かれ続けてきたように、ロゴは光と陰という最も根源的なコントラストだけで語ります。装飾を持たない分、情報量はゼロに近づき、意味の密度は極大に達します。

時代を超えて耐えうる設計 — それは「流行を取り込まない」ことと同義です。グラデーション、3D、ピクトグラム、アイソメトリック、ネオンスタイル。これらすべては一度は世界を席巻し、一度は古びました。IKELAB2026 のロゴはそれらの対極にあります。100年前の学術誌の扉絵に置いても違和感がなく、100年後の研究者の端末に表示されても恥ずかしくない。**変わらないためにこそ、選ばない**。

このロゴは、研究者の署名です。何を観測するかは時代によって変わるとしても、観測する者の姿勢は一定である — その誓いを形にしたものです。

## 02. ロゴタイプ仕様

| 項目 | 仕様 |
| :-- | :-- |
| 書体 | Instrument Sans Bold |
| トラッキング（字間） | −20 / 1000em |
| カラー | Pure Black `#000000` ／ Pure White `#FFFFFF` のいずれか |
| 大文字/小文字 | すべて大文字 |
| 比率 | ワードマーク幅 : 高さ ≒ 5 : 1（グリフバウンディングベース） |

### カラー定義

```
BLACK   RGB   0   0   0      HEX  #000000      CMYK  0  0  0  100
WHITE   RGB 255 255 255      HEX  #FFFFFF      CMYK  0  0  0    0
```

一色運用が絶対原則です。いかなるグラデーション・二色刷り・カラー化も行いません。

## 03. クリアスペース と 最小サイズ

### クリアスペース

ロゴの視認性と尊厳を守るため、**ワードマークの周囲に `2x` の保護領域**を必ず確保してください。

```
x = キャップハイト（大文字の高さ）÷ 2
```

この領域内には、他のテキスト・画像・罫線・装飾要素をいっさい配置しません。

### 最小サイズ

| 媒体 | 最小サイズ（横幅基準） |
| :-- | :-- |
| Webディスプレイ | **120 px** |
| 印刷物 | **20 mm** |

これ以下のサイズでの使用は、細部の破綻と誤認を招くため禁止します。

## 05. 運用ルール — Do / Don't

### ✅ Do（推奨される使用）

- 背景が白・薄いグレー・淡色の場合は **ブラック版** を使用する
- 背景が黒・濃色・写真上の暗部の場合は **ホワイト版** を使用する
- 周囲には `2x` 以上のクリアスペースを確保する
- 展開サイズに応じて最も近い配布サイズ（XL / L / M / S / XS）を選ぶ
- 印刷時は XL もしくはマスターデータから入稿する

### ❌ Don't（禁止される使用）

- ロゴを **回転・傾斜・歪曲** させること
- ロゴに **影・ぼかし・グラデーション・アウトライン** を加えること
- **指定外の色**（赤・青・金・その他）で着色すること
- 文字間隔や文字サイズを **個別に編集**すること
- 要素の一部（たとえば "IKELAB" だけ、"2026" だけ）を **切り取って使用**すること
- 背景写真のコントラストが十分でない場所に **可読性を損なう形で重ねる**こと
- 他のロゴ・マークと **近接配置**して混同を誘発すること
- 最小サイズを下回るサイズで使用すること

## 06. タイポグラフィの補助ルール（ロゴ外の文字組み）

ロゴと併記される書体は、視覚的一貫性のため以下を推奨します。

- **欧文（見出し・本文）**: Instrument Sans（Regular / Medium / Bold）
- **和文（見出し・本文）**: Noto Sans JP（Regular / Medium / Bold）

これらは無料で入手可能な Google Fonts のオープンソース書体であり、Web・印刷のいずれの媒体でも同一の見た目を担保します。
